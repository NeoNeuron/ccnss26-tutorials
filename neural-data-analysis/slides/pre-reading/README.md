# Pre-reading

Two short papers to read before the tutorials. PDFs included where freely available.

## Cunningham & Yu 2014
**Dimensionality reduction for large-scale neural recordings.** *Nature Neuroscience.*
- Key concept: population activity lies on a low-dimensional manifold; PCA + factor analysis are the workhorses.
- Read before **Session 2** (or both).
- [Author page](https://users.ece.cmu.edu/~byronyu/papers/CunninghamYuNN2014.pdf)

## Cohen & Kohn 2011
**Measuring and interpreting neuronal correlations.** *Nature Neuroscience.*
- Key concept: signal vs noise correlations; what correlated variability tells us about coding.
- Read before **Session 1**.
- [DOI](https://doi.org/10.1038/nn.2728) · [Author page — Kohn lab](https://www.cns.nyu.edu/~kohn/)
